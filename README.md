# R_Studio
R_Studio 학습 리포지터리
## 1일차 ~ 2일차
- 데이터 불러오기, 저장하기
- 데이터 정제하기
  - 파생변수 생성하기
  - 특정칼럼의 추출
  - 요약하기
- 결측치 처리하기

## 3일차
- 데이터 분석 진행하기
  - 실제데이터로 분석하기
  - 순서
    1. 데이터 확인(결측치, 이상치 확인)
    2. 결측치및 이상치 처리
    3. 데이터 정제(변수명등 변경)
    4. 분석하고자 하는 주제 세우기
    5. 설정한 주제와 관련된 데이터 수집, 셋팅
    6. 데이터 시각화(그래프)
- 텍스트 마이닝 설치
